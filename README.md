# STARMD

A WhatsApp Multi-Device Bot inspired by Anita-V5 but with different implementation.

## Features

- Multi-device support
- Group management
- Media download
- And more...

## Installation

1. Clone the repo
2. Install dependencies: `npm install`
3. Set up .env file
4. Run: `npm start`

## Deployment
Can be deployed on Heroku, Render, etc.
